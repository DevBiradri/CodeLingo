# CodeLingo Screen Specifications

Use these prompts with AI design tools (v0.dev, Midjourney, DALL-E) to generate UI components.

## 1. Welcome Hero Screen
**Prompt:** Mobile app UI, splash screen for a gamified coding app. 8-bit aesthetic mixed with modern glassmorphism. A central "Start Your Adventure" button in neon purple. Background features a blurred digital skill tree.

## 2. Choose Your Lore (Onboarding)
**Prompt:** Selection screen for a mobile game. Three vertical cards with glowing borders. Left card: Astronaut (Space). Middle card: Knight (Medieval). Right card: Hacker (Cyberpunk). Interactive UI, vibrant colors.

## 3. World Map (Skill Tree)
**Prompt:** Vertical scrolling map UI, Duolingo style but for a tech game. A path of circular icons winding through different biomes. Floating HUD at top showing 5 red hearts and a "15-day streak" flame icon.

## 4. Narrative Intro
**Prompt:** Visual novel style UI screen. A 2D stylized character speaking in a dialogue box. Background shows a futuristic server room. Narrative text: "The Grid is down, Hacker. We need to initialize the variables."

## 5. Parsons Problem (Drag-and-Drop)
**Prompt:** Mobile coding challenge UI. Top: Story text. Middle: Code blocks like LEGO bricks that can be dragged into a sequence. Neon green "Submit" button at the bottom. Dark mode theme.

## 6. AI Visualizer (ML Lab)
**Prompt:** Educational AI dashboard UI. A central scatter plot graph with a red/blue decision boundary line. A large slider at the bottom labeled "Adjust Synapse Weight." High-tech, laboratory aesthetic.

## 7. Victory Screen (Quest Complete)
**Prompt:** Victory screen for a mobile game. Confetti explosion. A large "Level 2" badge in the center. A golden progress bar filling up. Stats: "+20 XP", "100% Accuracy".
