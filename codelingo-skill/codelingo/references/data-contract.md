# CodeLingo Data Contract

## JSON Schema for Lessons
Every lesson generated by the AI must follow this structure to ensure compatibility across all roles.

```json
{
  "lesson_id": "string",
  "lore": "Cyberpunk | Space | Medieval",
  "narrative": "Story text introducing the lesson context.",
  "challenges": [
    {
      "id": "unique_challenge_id",
      "type": "parsons | fill_in_blank | multiple_choice | interactive_viz",
      "instruction": "Lore-specific instruction.",
      "technical_instruction": "Actual technical task (e.g., 'Declare a variable').",
      "blocks": ["array", "of", "code", "strings"], // For Parsons
      "code": "string with ____ for blanks", // For Fill-in-blank
      "options": ["list", "of", "choices"], // For Multiple Choice
      "solution": "the expected answer",
      "explanation": "AI-generated explanation for why the answer is correct/incorrect."
    }
  ],
  "theme_assets": {
    "primary_color": "hex",
    "character_sprite": "url/id"
  }
}
```
